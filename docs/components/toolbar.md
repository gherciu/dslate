## Toolbar

工具栏组件

### 用法

详情可见[@dslate/dslate](https://github.com/rojer95/dslate/blob/master/packages/dslate/src/components/DSlate/index.tsx)，这是对 Antd 风格的封装。

### 参数说明

| 参数    | 类型       | 说明                                                                  |
| ------- | ---------- | --------------------------------------------------------------------- |
| toolbar | `string[]` | 插件 `type` 数组，用于指定插件在工具栏展示的顺序，`divider`作为分隔栏 |
