# 题目编排

基于 DSlate 的 Antd 风格包，支持 html 渲染与小程序 RichNode 渲染模式

> ！！！重要提示！！！<br/>
> 1、小程序请手动引入 katex-mini 包的 css ，具体请看链接：https://github.com/rojer95/katex-mini <br/>
> 2、Latex 插件 从 DSlate V2.0.7 开始支持 <br/>
> 3、DSlate 2.0 起对 Antd 版本要求 >= 5.0
